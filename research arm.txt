The Shadow Model: A Hybrid Intelligence Engine for Unsupervised Discovery of Novel Psychological Constructs
Abstract
This thesis details the architecture and methodology of the Shadow Model, a core component of the Consciousness Archaeology Engine (CAE). The Shadow Model is designed to transcend the limitations of existing psychometric frameworks by autonomously discovering novel, emergent psychological constructs from raw behavioral data. It operates as a hybrid intelligence, combining the quantitative rigor of unsupervised machine learning for pattern detection with the qualitative, inferential reasoning of a collaborative agentic "ThinkTank" for sense-making. The model processes high-dimensional linguistic embeddings of user responses through a UMAP and HDBSCAN pipeline to identify statistically significant behavioral clusters. Novel clusters are then passed as a briefing to a multi-agent ThinkTank, which collaborates to hypothesize, define, and critique the potential new trait, culminating in a formal research report. This report is then presented to human users via a "Research Portal" for collaborative validation, creating a continuous, Human-AI feedback loop that allows the CAE to evolve its own understanding of the human psyche.





1.0 Introduction: The Problem of Pre-defined Frameworks
Traditional psychological assessments, while valuable, are inherently constrained by the theories upon which they are built, such as the Big Five or HEXACO. They measure individuals against pre-defined traits, potentially missing the nuanced, emergent, or culturally specific ways in which personality manifests. This creates a "streetlight effect," where we only look for psychological phenomena where the theoretical light is already shining.

The core challenge the Shadow Model addresses is: How can an AI system discover meaningful, latent psychological patterns from behavioral data without being biased by a priori frameworks?. The solution is a two-stage discovery engine that first identifies statistically robust patterns and then uses sophisticated agentic reasoning to interpret them.


2.0 Architectural Placement and Function
The Shadow Model is not part of the real-time conversational loop. It operates as an asynchronous, computationally intensive process that runs in the background, analyzing the aggregated data collected by the CAE.


Input: The model ingests a large collection of LinguisticSignature objects generated from user responses, specifically focusing on the high-dimensional embedding vectors.

Primary Output: The process culminates in a ClusterAnalysis object for each novel pattern discovered. This object is a structured data packet containing the cluster's centroid, member response IDs, and statistical properties. This object serves as the initial briefing document for the next stage of the pipeline: the Discovery ThinkTank.

3.0 Stage 1: The Algorithmic Signal Detector
The first stage of the Shadow Model is a purely algorithmic pipeline designed to find statistically significant "signals" in the noise of conversational data.

3.1 Dimensionality Reduction with UMAP: The raw embedding vectors from the LinguisticSignature objects are high-dimensional. To make clustering computationally feasible and effective, the model first uses Uniform Manifold Approximation and Projection (UMAP) to reduce the dimensionality (e.g., to 10D). UMAP is chosen over other methods like PCA because it excels at preserving the data's topological structure, meaning that points that are "close" in the high-dimensional space remain close in the reduced space.



3.2 Clustering with HDBSCAN: The reduced-dimension embeddings are then fed into the Hierarchical Density-Based Spatial Clustering of Applications with Noise (HDBSCAN) algorithm. HDBSCAN is superior to simpler algorithms like K-Means for this task because it does not assume clusters are spherical and, crucially, it can identify points as "noise," effectively ignoring outliers that do not belong to any coherent behavioral pattern.




3.3 Novelty Validation: Once a cluster is identified, it must be verified as genuinely novel. The system calculates the cluster's centroid embedding and computes its cosine similarity to the centroids of all known traits in the Psycho-Analytic Knowledge Graph (PKG) and any previously discovered latent traits. If the similarity to any existing trait is above a certain threshold (e.g., > 0.8), the cluster is marked as a known construct and is not passed on for further analysis.



4.0 Stage 2: The Agentic ThinkTank for Sense-Making
A statistically novel cluster is a signal, but it is not yet an insight. The ClusterAnalysis object is passed to a dedicated, asynchronous "Discovery ThinkTank" to perform the interpretive work of turning a pattern into a hypothesis.

4.1 Collaborative Hypothesis Generation: The ThinkTank is composed of specialized AI agents, each tasked with a specific cognitive function:

The Analyst Agent: Reviews the exemplar user responses within the cluster to provide a qualitative summary of common linguistic themes, emotional tones, and concepts.
The Namer Agent: Proposes several candidate names for the new construct based on the Analyst's summary.
The Definer Agent: Drafts a formal, falsifiable scientific definition for the proposed trait, aiming for precision and clarity.


The Critic Agent: The cornerstone of the crew's intellectual rigor. The Critic's function is to actively challenge the hypothesis. It cross-references the proposed definition against the PKG to find potential overlaps, points out logical fallacies or ambiguities, and forces the other agents to refine their work.
4.2 Internal Debate and Refinement: The agents engage in a recursive loop of proposing, critiquing, and refining their outputs until a stable consensus is reached. This internal dialectic ensures the final hypothesis is well-considered and robust.

4.3 Output: The Research Report: The ThinkTank's final product is a structured research report containing the proposed trait name, its formal definition, a confidence score, the supporting linguistic evidence, and a summary of the counter-arguments considered by the Critic.


5.0 Stage 3: Human-in-the-Loop Collaborative Validation
The AI-generated research report is a hypothesis, not a conclusion. It is published to a "Research Portal" within the CAE, inviting the user community to participate in the final stage of validation.

Community Feedback: Users can vote on the best name for the new trait, provide qualitative feedback on whether the definition accurately captures the essence of the exemplar responses, and engage in discussion.
Experimental Validation: Most importantly, the ThinkTank's report includes a proposal for a new, targeted question designed to specifically probe for the hypothesized trait. Users can opt-in to "research sessions" where the CAE deploys these experimental probes. The system then measures if a user's response to the new probe is a strong predictor of their membership in the original cluster, thus validating the trait's existence and the probe's effectiveness.
6.0 Closing the Loop: Integrating Discovery into the Live System
The discovery process is not isolated. Once a new trait has achieved a sufficient level of validation via the Research Portal, it is formally added to the CAE's Psycho-Analytic Knowledge Graph (PKG).

This has a profound impact on the real-time conversational engine:

The newly validated trait becomes a potential target_concept for the AdaptiveSelector.

The live CAE can now intelligently decide to probe for this new trait, using the AI-generated and community-validated questions to gather more data on it.
This closes the loop, turning the Shadow Model's discoveries into active tools for deeper, more nuanced psychological assessment in every subsequent conversation.

import spacy
import numpy as np
from transformers import AutoTokenizer, AutoModel
from textstat import flesch_reading_ease, flesch_kincaid_grade
from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer
from core_models import LinguisticSignature
import asyncio
import time
import re
import torch

class LinguisticSignatureExtractor:
    def __init__(self, transformer_model: str = "distilbert-base-uncased"):
        self.nlp = spacy.load("en_core_web_lg")
        self.sentiment_analyzer = SentimentIntensityAnalyzer()
        self.tokenizer = AutoTokenizer.from_pretrained(transformer_model)
        self.transformer = AutoModel.from_pretrained(transformer_model)
        self.transformer.eval()
        self.psychological_themes = [
            "achievement", "social_relationships", "emotional_expression",
            "risk_uncertainty", "control_agency", "creativity_imagination",
            "moral_reasoning", "temporal_orientation", "abstract_thinking"
        ]

    async def extract(self, text: str) -> LinguisticSignature:
        start_time = time.time()
        
        # Parallel analysis tasks
        structural_task = asyncio.create_task(self._analyze_structure(text))
        semantic_task = asyncio.create_task(self._analyze_semantics(text))
        stance_task = asyncio.create_task(self._analyze_stance(text))
        embedding_task = asyncio.create_task(self._generate_embedding(text))
        
        structural, semantic, stance, embedding = await asyncio.gather(
            structural_task, semantic_task, stance_task, embedding_task
        )
        
        processing_time = time.time() - start_time
        confidence = self._compute_confidence(text, structural, semantic, stance)
        
        return LinguisticSignature(
            structural_features=structural,
            semantic_features=semantic,
            stance_features=stance,
            embedding=embedding,
            confidence=confidence,
            processing_time=processing_time
        )

    async def _analyze_structure(self, text: str) -> dict:
        doc = self.nlp(text)
        word_count = len([t for t in doc if not t.is_punct])
        sentence_count = len(list(doc.sents))
        unique_words = len(set(t.lemma_.lower() for t in doc if t.is_alpha))
        type_token_ratio = unique_words / max(word_count, 1)
        
        dependency_depths = [self._get_dependency_depth(t) for t in doc]
        avg_dep_depth = np.mean(dependency_depths) if dependency_depths else 0
        max_dep_depth = max(dependency_depths) if dependency_depths else 0
        
        pos_tags = [t.pos_ for t in doc if t.is_alpha]
        pos_diversity = len(set(pos_tags)) / max(len(pos_tags), 1)
        
        flesch_ease = flesch_reading_ease(text)
        flesch_grade = flesch_kincaid_grade(text)
        subordinate_clauses = len([t for t in doc if t.dep_ in ['ccomp', 'xcomp', 'advcl']])
        subordination_ratio = subordinate_clauses / max(sentence_count, 1)
        
        return {
            "word_count": float(word_count),
            "sentence_count": float(sentence_count),
            "type_token_ratio": type_token_ratio,
            "avg_dependency_depth": avg_dep_depth,
            "max_dependency_depth": float(max_dep_depth),
            "pos_diversity": pos_diversity,
            "flesch_reading_ease": flesch_ease,
            "flesch_kincaid_grade": flesch_grade,
            "subordination_ratio": subordination_ratio
        }

    async def _analyze_semantics(self, text: str) -> dict:
        sentiment_scores = self.sentiment_analyzer.polarity_scores(text)
        doc = self.nlp(text)
        
        emotional_words = [t for t in doc if any(emo in t.text.lower() for emo in 
                          ['feel', 'emotion', 'happy', 'sad', 'angry', 'fear', 'joy'])]
        emotional_density = len(emotional_words) / max(len(doc), 1)
        
        abstract_patterns = ['concept', 'idea', 'notion', 'principle', 'theory']
        concrete_patterns = ['see', 'hear', 'touch', 'smell', 'taste', 'object']
        abstract_count = sum(1 for p in abstract_patterns if p in text.lower())
        concrete_count = sum(1 for p in concrete_patterns if p in text.lower())
        abstraction_ratio = abstract_count / max(abstract_count + concrete_count, 1)
        
        metaphor_patterns = [r'like a', r'as if', r'reminds me of', r'similar to']
        metaphor_count = sum(len(re.findall(p, text, re.IGNORECASE)) for p in metaphor_patterns)
        metaphor_density = metaphor_count / max(len(text.split()), 1)
        
        return {
            "sentiment_positive": sentiment_scores['pos'],
            "sentiment_negative": sentiment_scores['neg'],
            "sentiment_neutral": sentiment_scores['neu'],
            "sentiment_compound": sentiment_scores['compound'],
            "emotional_density": emotional_density,
            "abstraction_ratio": abstraction_ratio,
            "metaphor_density": metaphor_density
        }

    async def _analyze_stance(self, text: str) -> dict:
        doc = self.nlp(text)
        first_person = len([t for t in doc if t.text.lower() in ['i', 'me', 'my', 'myself']])
        third_person = len([t for t in doc if t.text.lower() in ['they', 'them', 'their']])
        total_pronouns = first_person + third_person
        agency_ratio = first_person / max(total_pronouns, 1)
        
        certainty_words = ['definitely', 'certainly', 'absolutely', 'sure', 'confident']
        uncertainty_words = ['maybe', 'perhaps', 'possibly', 'might', 'could', 'uncertain']
        certainty_count = sum(1 for w in certainty_words if w in text.lower())
        uncertainty_count = sum(1 for w in uncertainty_words if w in text.lower())
        certainty_ratio = certainty_count / max(certainty_count + uncertainty_count, 1)
        
        past_tense = len([t for t in doc if t.tag_ in ['VBD', 'VBN']])
        present_tense = len([t for t in doc if t.tag_ in ['VBZ', 'VBP']])
        future_tense = len([t for t in doc if 'will' in t.text.lower() or 'going to' in text.lower()])
        total_verbs = past_tense + present_tense + future_tense
        temporal_focus = {
            "past_orientation": past_tense / max(total_verbs, 1),
            "present_orientation": present_tense / max(total_verbs, 1),
            "future_orientation": future_tense / max(total_verbs, 1)
        }
        
        modal_verbs = ['can', 'could', 'may', 'might', 'must', 'should', 'would', 'will']
        modal_count = sum(1 for t in doc if t.text.lower() in modal_verbs)
        modal_density = modal_count / max(len(doc), 1)
        
        question_count = text.count('?')
        sentence_count = len(list(doc.sents))
        question_ratio = question_count / max(sentence_count, 1)
        
        return {
            "agency_ratio": agency_ratio,
            "certainty_ratio": certainty_ratio,
            "modal_density": modal_density,
            "question_ratio": question_ratio,
            **temporal_focus
        }

    async def _generate_embedding(self, text: str) -> list:
        inputs = self.tokenizer(text, return_tensors="pt", truncation=True, max_length=512)
        with torch.no_grad():
            outputs = self.transformer(**inputs)
            embedding = outputs.last_hidden_state[:, 0, :].squeeze().numpy()
        return embedding.tolist()

    def _get_dependency_depth(self, token) -> int:
        depth = 0
        current = token
        while current.head != current:
            depth += 1
            current = current.head
        return depth

    def _compute_confidence(self, text: str, structural: dict, semantic: dict, stance: dict) -> float:
        word_count = structural.get("word_count", 0)
        length_confidence = min(word_count / 50.0, 1.0)  # Optimal at 50+ words
        if word_count < 10:
            length_confidence *= 0.5
        total_features = len(structural) + len(semantic) + len(stance)
        expected_features = 25
        completeness = min(total_features / expected_features, 1.0)
        return (length_confidence + completeness) / 2.0
Features: Structural (syntax, readability), semantic (sentiment, themes), stance (agency, certainty), and dense embeddings (DistilBERT for now, Mamba-2 later).
Alignment: Extracts features for IPIP-300 facets (e.g., Sociability: “Feel comfortable around people”) and high-alpha items (e.g., “Talk to a lot of different people at parties,” α = 0.87).

from transformers import MambaForCausalLM, AutoTokenizer
import torch
import numpy as np
from core_models import LinguisticSignature

class MambaSemanticMapper:
    def __init__(self, model_name: str = "mamba-2", n_traits: int = 300):
        self.model = MambaForCausalLM.from_pretrained(model_name)
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.projection_layer = torch.nn.Linear(768, n_traits)  # Map embeddings to trait scores
        self.n_traits = n_traits

    def map_to_traits(self, signature: LinguisticSignature) -> np.ndarray:
        # Convert embedding to tensor
        embedding = torch.tensor(signature.embedding, dtype=torch.float32).unsqueeze(0)
        
        # Generate contextualized representation
        prompt = f"Analyze this response for personality traits: {signature.response_text}"
        inputs = self.tokenizer(prompt, return_tensors="pt")
        with torch.no_grad():
            mamba_output = self.model(**inputs).hidden_states[-1][:, 0, :]  # Last state
            combined = torch.cat((embedding, mamba_output), dim=1)
            trait_scores = self.projection_layer(combined).squeeze().numpy()
        
        return np.clip(trait_scores, 1, 5)  # IPIP-300 scores (1–5)

    def train(self, signatures: list[LinguisticSignature], ground_truth: list[np.ndarray]):
        optimizer = torch.optim.Adam(self.projection_layer.parameters(), lr=1e-3)
        loss_fn = torch.nn.MSELoss()
        
        for epoch in range(10):
            for sig, gt in zip(signatures, ground_truth):
                embedding = torch.tensor(sig.embedding, dtype=torch.float32).unsqueeze(0)
                prompt = f"Analyze this response for personality traits: {sig.response_text}"
                inputs = self.tokenizer(prompt, return_tensors="pt")
                with torch.no_grad():
                    mamba_output = self.model(**inputs).hidden_states[-1][:, 0, :]
                combined = torch.cat((embedding, mamba_output), dim=1)
                pred_scores = self.projection_layer(combined).squeeze()
                gt_tensor = torch.tensor(gt, dtype=torch.float32)
                
                loss = loss_fn(pred_scores, gt_tensor)
                optimizer.zero_grad()
                loss.backward()
                optimizer.step()
Mechanism: Mamba-2 contextualizes response embeddings with prompt-based reasoning, projecting to IPIP-300 scores or cluster beliefs. Selective attention (S6) prioritizes personality-relevant features (e.g., agency for Sociability).
Training: Pre-train on clinical transcripts (CDI-style) and fine-tune on simulated responses for IPIP-300 items (e.g., “I mingled with everyone” → Sociability=5) and high-alpha items (e.g., “Am exacting in my work,” α = 0.81).

import umap
import hdbscan
import numpy as np
from uuid import uuid4
from datetime import datetime
from core_models import LinguisticSignature, LatentTrait, ClusterAnalysis
from transformers import MambaForCausalLM, AutoTokenizer

class ShadowModel:
    def __init__(self, min_cluster_size: int = 5, min_samples: int = 3):
        self.min_cluster_size = min_cluster_size
        self.min_samples = min_samples
        self.reducer = umap.UMAP(n_components=10, n_neighbors=15, min_dist=0.1, metric='cosine')
        self.clusterer = hdbscan.HDBSCAN(min_cluster_size=min_cluster_size, 
                                         min_samples=min_samples, metric='euclidean')
        self.mamba = MambaForCausalLM.from_pretrained("mamba-2")
        self.tokenizer = AutoTokenizer.from_pretrained("mamba-2")
        self.discovered_traits = {}

    async def process(self, signatures: list[LinguisticSignature], 
                     response_texts: list[str]) -> list[ClusterAnalysis]:
        embeddings = np.array([sig.embedding for sig in signatures])
        if len(embeddings) < self.min_cluster_size * 2:
            return []
        
        # Dimensionality reduction
        reduced_embeddings = self.reducer.fit_transform(embeddings)
        
        # Clustering
        cluster_labels = self.clusterer.fit_predict(reduced_embeddings)
        cluster_analyses = []
        
        for label in set(cluster_labels):
            if label == -1:  # Noise
                continue
            mask = cluster_labels == label
            cluster_indices = np.where(mask)[0]
            if len(cluster_indices) < self.min_cluster_size:
                continue
            
            cluster_embeddings = reduced_embeddings[mask]
            centroid = np.mean(cluster_embeddings, axis=0)
            intra_distance = np.mean(np.linalg.norm(cluster_embeddings - centroid, axis=1))
            silhouette = self._compute_silhouette_score(reduced_embeddings, cluster_labels, label)
            
            analysis = ClusterAnalysis(
                cluster_id=int(label),
                response_ids=[uuid4() for _ in cluster_indices],  # Placeholder IDs
                centroid_embedding=centroid.tolist(),
                intra_cluster_distance=float(intra_distance),
                silhouette_score=float(silhouette),
                is_novel=self._is_novel_cluster(centroid)
            )
            cluster_analyses.append(analysis)
            
            # Generate trait hypothesis for novel clusters
            if analysis.is_novel and analysis.silhouette_score > 0.3:
                trait = await self._generate_trait_hypothesis(analysis, response_texts, cluster_indices)
                if trait:
                    self.discovered_traits[trait.id] = trait
        
        return cluster_analyses

    async def _generate_trait_hypothesis(self, cluster: ClusterAnalysis, 
                                        responses: list[str], 
                                        indices: list[int]) -> LatentTrait:
        exemplars = [responses[i] for i in indices[:3]]
        anti_exemplars = self._extract_anti_exemplars(cluster, responses, indices)
        
        prompt = f"""Analyze these responses for a latent personality trait:
        CORE: {chr(10).join(f'- {e}' for e in exemplars)}
        CONTRAST: {chr(10).join(f'- {a}' for a in anti_exemplars)}
        
        Requirements:
        1. Name: Single phrase
        2. Definition: Scientific, falsifiable (2-3 sentences)
        3. Confidence: 0–1 score
        
        Output:
        Name: [trait name]
        Definition: [definition]
        Confidence: [score]"""
        
        inputs = self.tokenizer(prompt, return_tensors="pt")
        with torch.no_grad():
            output = self.mamba.generate(**inputs, max_length=200)
        result = self.tokenizer.decode(output[0], skip_special_tokens=True)
        
        # Parse output (simplified)
        lines = result.split('\n')
        name = lines[0].replace("Name: ", "").strip()
        definition = lines[1].replace("Definition: ", "").strip()
        confidence = float(lines[2].replace("Confidence: ", "").strip())
        
        return LatentTrait(
            id=uuid4(),
            name=name,
            definition=definition,
            exemplar_responses=exemplars,
            anti_exemplar_responses=anti_exemplars,
            validation_status="hypothesized",
            discovery_date=datetime.utcnow(),
            confidence_score=confidence,
            supporting_evidence_count=len(exemplars)
        )

    def _extract_anti_exemplars(self, cluster: ClusterAnalysis, 
                               responses: list[str], 
                               cluster_indices: list[int]) -> list[str]:
        all_indices = set(range(len(responses))) - set(cluster_indices)
        if not all_indices:
            return []
        
        centroid = np.array(cluster.centroid_embedding)
        reduced_embeddings = self.reducer.transform(
            np.array([self._get_embedding(r) for r in responses])
        )
        
        distances = [np.linalg.norm(reduced_embeddings[i] - centroid) 
                     for i in all_indices]
        boundary_distance = cluster.intra_cluster_distance * 1.5
        boundary_candidates = [(i, d) for i, d in zip(all_indices, distances) 
                              if boundary_distance * 0.8 <= d <= boundary_distance * 2.0]
        
        boundary_candidates.sort(key=lambda x: x[1])
        return [responses[i] for i, _ in boundary_candidates[:3]]

    def _compute_silhouette_score(self, embeddings: np.ndarray, 
                                 labels: np.ndarray, 
                                 cluster_label: int) -> float:
        cluster_mask = labels == cluster_label
        cluster_points = embeddings[cluster_mask]
        if len(cluster_points) <= 1:
            return 0.0
        
        intra_distances = [np.mean(np.linalg.norm(cluster_points - p, axis=1)) 
                          for p in cluster_points]
        inter_distances = []
        other_clusters = set(labels) - {cluster_label, -1}
        
        for point in cluster_points:
            min_inter = float('inf')
            for other_label in other_clusters:
                other_mask = labels == other_label
                distances = np.linalg.norm(embeddings[other_mask] - point, axis=1)
                if len(distances) > 0:
                    min_inter = min(min_inter, np.mean(distances))
            inter_distances.append(min_inter)
        
        silhouette_scores = [(b - a) / max(a, b) 
                            for a, b in zip(intra_distances, inter_distances) 
                            if max(a, b) > 0]
        return np.mean(silhouette_scores) if silhouette_scores else 0.0

    def _is_novel_cluster(self, centroid: np.ndarray) -> bool:
        for trait in self.discovered_traits.values():
            if hasattr(trait, 'centroid_embedding'):
                existing = np.array(trait.centroid_embedding)
                similarity = np.dot(centroid, existing) / (
                    np.linalg.norm(centroid) * np.linalg.norm(existing)
                )
                if similarity > 0.8:
                    return False
        return True

    def _get_embedding(self, text: str) -> np.ndarray:
        # Placeholder: Use DistilBERT for now, replace with Mamba-2 later
        inputs = self.tokenizer(text, return_tensors="pt", truncation=True, max_length=512)
        with torch.no_grad():
            outputs = self.transformer(**inputs)
            embedding = outputs.last_hidden_state[:, 0, :].squeeze().numpy()
        return embedding
Mechanism: Clusters response embeddings (UMAP to 10D, HDBSCAN for flexible shapes) and generates trait hypotheses for novel clusters (silhouette > 0.3) using Mamba-2’s generative capabilities.
Role: Discovers constructs beyond IPIP-300/Big Five (e.g., “Conceptual Abstraction” from high abstract_thinking responses). Hypotheses are validated via targeted probes (e.g., “Describe a time you solved a problem creatively”).
Training: Pre-train Mamba-2 on clinical transcripts and fine-tune on clustered items from “full personality.txt” (e.g., “Am an original thinker,” α = 0.85) to generate falsifiable definitions.
Output: LatentTrait objects (e.g., Name: “Temporal Projection,” Definition: “Bias toward future-focused thinking,” Confidence: 0.7).

        embeddings = []
        for text in high_alpha_items['text']:
            inputs = self.tokenizer(text, return_tensors="pt", truncation=True, max_length=512)
