Key Optimizations
HEDGES Integration:
Purpose: HEDGES corrects insertions, deletions, and substitutions within a single read, achieving 1.3 b/nt payload-only density and 1.0 b/nt overall density with ~5 coverage (dna_info.pdf, p. 27). It supports scalable, error-resilient DNA storage.
Implementation: Enhance DNASpec and SynapticDEWE to implement HEDGES with a Reed-Solomon outer code and hash-based inner code, using a greedy search decoder. Incorporate interleavers for homopolymer control (dna_info.pdf, p. 24).
Savanna Framework:
Features Utilized: Savanna’s support for StripedHyena 2 (multi-hybrid architecture with Hyena-SE, Hyena-MR, Hyena-LI operators), byte-level tokenization, DeepSpeed ZeRO sharding, and context parallelism (a2a, p2p) will optimize distributed finetuning (Savanna description).
Implementation: Configure MSDEGA to use Savanna’s launch.py with custom configs for data (data/savannah.yml) and model (model/evo2_1b.yml). Enable FP8 precision via TransformerEngine for efficiency on A100 GPUs.
Evo-2 1B Base Model:
Rationale: The Evo-2 1B model’s smaller size (1.1B parameters, ~4GB VRAM) and limited pretraining (1T tokens, evo2.pdf, Table 2) make its weights more responsive to finetuning, ideal for DNA encoding tasks.
Finetuning Strategy: Use Savanna’s reweighted cross-entropy loss (evo2.pdf, p. 19) with weights adjusted for repetitive DNA regions (0.1) and a custom loss incorporating DNA encoding metrics (info_density, error_rate, random_access, constraints).
Synthetic Savannah Dataset:
Assumption: The Savannah dataset is a synthetic genomic dataset with 10,000 sequences (50–300 nt each), mimicking OpenGenome2’s diversity (evo2.pdf, p. 4). It includes prokaryotic and eukaryotic sequences with functional annotations.
Implementation: Modify DNADataset to generate sequences compatible with HEDGES and Evo-2’s byte-level tokenization, ensuring alignment with Savanna’s data pipeline.
Computational Efficiency:
Hardware: 4 A100 GPUs (8GB each) for primary finetuning, with fallback to RTX 3060 (12GB VRAM) for local deployment. Savanna’s ZeRO sharding and context parallelism optimize memory usage.
Optimization: Set graph sparsity to 20%, use batch sizes tailored for A100s, and leverage StripedHyena 2’s 1.3× speedup at 16k context (evo2.pdf, p. 4).
Optimized MSDEGA Implementation
Below is the refined MSDEGA code, optimized for finetuning Evo-2 1B using Savanna with HEDGES compression. The implementation integrates Savanna’s training configs, HEDGES encoding, and aligns with the Life as an Adaptive Relational Web thesis.

class DNASpec:
    """HEDGES-based DNA encoding specification (dna_info.pdf, p. 27)"""
    def __init__(self, config):
        self.gc_min = config.gc_min
        self.gc_max = config.gc_max
        self.max_homopolymer = config.max_homopolymer
        self.target_density = config.target_density
        self.primer_length = config.primer_length
        self.error_correction_method = config.error_correction_method
        self.interleavers = [
            lambda x: x,
            lambda x: x[::-1],
            lambda x: ''.join([x[i] for i in range(0, len(x), 2)] + [x[i] for i in range(1, len(x), 2)]),
            lambda x: ''.join([x[i] for i in range(0, len(x), 3)] + [x[i] for i in range(1, len(x), 3)] + [x[i] for i in range(2, len(x), 3)])
        ]
        self.salt_bits = 8  # HEDGES salt (dna_info.pdf, p. 25)
        logger.info(f"Initialized HEDGES DNA spec: GC={self.gc_min}-{self.gc_max}, max homopolymer={self.max_homopolymer}")

    def validate_sequence(self, sequence):
        if not sequence or not all(c in 'ACGT' for c in sequence):
            return False
        gc_count = sum(1 for c in sequence if c in 'GC')
        gc_ratio = gc_count / len(sequence)
        if not (self.gc_min <= gc_ratio <= self.gc_max):
            return False
        prev_nt = None
        run_length = 1
        for nt in sequence:
            if nt == prev_nt:
                run_length += 1
                if run_length > self.max_homopolymer:
                    return False
            else:
                run_length = 1
            prev_nt = nt
        return True

    def fix_sequence(self, sequence):
        if not sequence:
            return ""
        sequence = ''.join(c if c in 'ACGT' else np.random.choice(['A', 'C', 'G', 'T']) for c in sequence)
        fixed_seq = ""
        prev_nt = None
        run_length = 1
        for i, nt in enumerate(sequence):
            if nt == prev_nt:
                run_length += 1
                if run_length > self.max_homopolymer:
                    replacements = [n for n in 'ACGT' if n != prev_nt]
                    nt = replacements[i % len(replacements)]
                    run_length = 1
            else:
                run_length = 1
            fixed_seq += nt
            prev_nt = nt
        gc_count = sum(1 for c in fixed_seq if c in 'GC')
        gc_ratio = gc_count / len(fixed_seq)
        if gc_ratio < self.gc_min:
            needed = int((self.gc_min - gc_ratio) * len(fixed_seq)) + 1
            at_positions = [i for i, c in enumerate(fixed_seq) if c in 'AT']
            if at_positions:
                to_replace = np.random.choice(at_positions, size=min(needed, len(at_positions)), replace=False)
                fixed_seq = list(fixed_seq)
                for pos in to_replace:
                    fixed_seq[pos] = np.random.choice(['G', 'C'])
                fixed_seq = ''.join(fixed_seq)
        elif gc_ratio > self.gc_max:
            needed = int((gc_ratio - self.gc_max) * len(fixed_seq)) + 1
            gc_positions = [i for i, c in enumerate(fixed_seq) if c in 'GC']
            if gc_positions:
                to_replace = np.random.choice(gc_positions, size=min(needed, len(gc_positions)), replace=False)
                fixed_seq = list(fixed_seq)
                for pos in to_replace:
                    fixed_seq[pos] = np.random.choice(['A', 'T'])
                fixed_seq = ''.join(fixed_seq)
        for interleaver in self.interleavers:
            interleaved = interleaver(fixed_seq)
            if self.validate_sequence(interleaved):
                return interleaved
        return fixed_seq

    def hedges_encode(self, bits, packet_id, seq_id, prev_bits=4):
        """HEDGES encoding with hash-based inner code (dna_info.pdf, p. 25)"""
        salt = np.random.randint(0, 2**self.salt_bits, dtype=np.uint8).tobytes()
        sequence = ""
        for i, bit in enumerate(bits):
            index = i.to_bytes(4, 'big')
            prev = bits[max(0, i-prev_bits):i].tobytes() if i > 0 else b''
            hash_input = salt + index + prev
            hash_val = int(sha256(hash_input).hexdigest(), 16) % 4
            encoded = (bit + hash_val) % 4
            sequence += ['A', 'C', 'G', 'T'][encoded]
        primer_5 = ''.join(np.random.choice(['A', 'C', 'G', 'T'], self.primer_length))
        primer_3 = ''.join(np.random.choice(['A', 'C', 'G', 'T'], self.primer_length))
        sequence = primer_5 + sequence + primer_3
        if not self.validate_sequence(sequence):
            sequence = self.fix_sequence(sequence)
        return sequence

    def hedges_decode(self, sequence):
        """Simplified HEDGES greedy search decoding (dna_info.pdf, p. 26)"""
        if len(sequence) < 2 * self.primer_length:
            return []
        payload = sequence[self.primer_length:-self.primer_length]
        bits = []
        for i, nt in enumerate(payload):
            bit_guess = ['A', 'C', 'G', 'T'].index(nt) % 2  # Simplified decoding
            bits.append(bit_guess)
        return bits

class SynapticGraphNet(MessagePassing):
    """Graph neural network for synaptic processing, inspired by GMNs"""
    def __init__(self, in_channels, out_channels):
        super().__init__(aggr='add')
        self.lin = nn.Linear(in_channels * 2, out_channels)
        self.act = nn.ReLU()

    def forward(self, x, edge_index, edge_attr):
        return self.propagate(edge_index, x=x, edge_attr=edge_attr)

    def message(self, x_i, x_j, edge_attr):
        return self.act(self.lin(torch.cat([x_i, edge_attr], dim=-1)))

class SynapticDEWE(nn.Module):
    """Synaptic DNA encoding variant for Evo-2 1B with HEDGES"""
    def __init__(self, config):
        super().__init__()
        self.config = config
        self.dna_spec = DNASpec(config)
        self.graph_net = SynapticGraphNet(64, 64).to(device)
        self.evo_graph = nx.Graph()
        self.model = nn.Sequential(
            nn.Linear(64, 128),
            nn.ReLU(),
            nn.Linear(128, 64)
        ).to(device)
        self.meta_optimizer = optim.Adam(self.parameters(), lr=config.variant_configs["SynapticDEWE"]["meta_lr"])
        self.hyper_optimizer = optim.Adam([torch.tensor(config.variant_configs["SynapticDEWE"]["inner_lr_init"], requires_grad=True, device=device)], lr=config.variant_configs["SynapticDEWE"]["hyper_lr"])
        self.best_fitness = -float('inf')
        self.best_solution = None
        self.iteration = 0
        self.name = "SynapticDEWE"
        self.initialize()

    def initialize(self):
        self.evo_graph.add_nodes_from(range(self.config.synaptic_nodes), type="synaptic", state=torch.randn(64, device=device))
        for i in range(self.config.synaptic_nodes):
            for j in range(i + 1, self.config.synaptic_nodes):
                if np.random.random() < self.config.sparsity:
                    self.evo_graph.add_edge(i, j, weight=np.random.uniform(0.1, 0.5))
        logger.info(f"Initialized {self.name} with {self.config.synaptic_nodes} nodes")

    def run_iteration(self, compute_allocation):
        generations = max(1, int(compute_allocation * self.config.variant_configs["SynapticDEWE"]["generations"] / self.config.total_compute_per_iteration))
        dataset = DNADataset(self.config)
        node_features = torch.stack([self.evo_graph.nodes[i]['state'] for i in range(self.config.synaptic_nodes)]).to(device)
        edge_index = torch.tensor([[u, v] for u, v in self.evo_graph.edges()], dtype=torch.long).t().to(device)
        edge_attr = torch.tensor([self.evo_graph[u][v]['weight'] for u, v in self.evo_graph.edges()], dtype=torch.float).unsqueeze(-1).to(device)
        metrics_history = {m: [] for m in self.config.metrics}
        fitness_scores = []

        for gen in range(generations):
            tasks = [dataset[i] for i in np.random.choice(len(dataset), size=min(len(dataset), self.config.variant_configs["SynapticDEWE"]["task_batch_size"]), replace=False)]
            metrics_batch = []
            for _ in range(self.config.variant_configs["SynapticDEWE"]["population_size"]):
                fast_weights, task_loss = self.adapt_to_task(tasks[0], node_features, edge_index, edge_attr)
                node_features_new = self.graph_net(node_features, edge_index, edge_attr)
                metrics = self.compute_metrics(node_features_new, edge_index, edge_attr, tasks)
                fitness = self.calculate_fitness(metrics)
                fitness_scores.append(fitness)
                metrics_batch.append(metrics)
            self.evolve_graph(fitness_scores)
            node_features = self.graph_net(node_features, edge_index, edge_attr)
            avg_metrics = self.average_metrics(metrics_batch)
            for metric in metrics_history:
                metrics_history[metric].append(avg_metrics[metric])
            meta_loss = sum(self.task_loss(task, node_features, edge_index, edge_attr) for task in tasks)
            self.meta_optimizer.zero_grad()
            if meta_loss.requires_grad:
                meta_loss.backward()
                self.meta_optimizer.step()
            self.hyper_optimizer.zero_grad()
            if meta_loss.requires_grad:
                meta_loss.backward()
                self.hyper_optimizer.step()
        result = {
            "fitness": max(fitness_scores),
            "compute_used": generations,
            "metrics": avg_metrics,
            "metrics_history": metrics_history
        }
        if result["fitness"] > self.best_fitness:
            self.best_fitness = result["fitness"]
            self.best_solution = result
        return result

    def adapt_to_task(self, task, node_features, edge_index, edge_attr):
        fast_weights = {name: param.clone() for name, param in self.graph_net.named_parameters()}
        for _ in range(self.config.variant_configs["SynapticDEWE"]["inner_steps"]):
            node_features_new = self.graph_net(node_features, edge_index, edge_attr)
            loss = self.task_loss(task, node_features_new, edge_index, edge_attr)
            grads = torch.autograd.grad(loss, fast_weights.values(), create_graph=True, allow_unused=True)
            fast_weights = {
                name: param - self.config.variant_configs["SynapticDEWE"]["inner_lr_init"] * (grad if grad is not None else torch.zeros_like(param))
                for (name, param), grad in zip(fast_weights.items(), grads)
            }
        return fast_weights, loss.item()

    def task_loss(self, task, node_features, edge_index, edge_attr):
        task_type = task["type"]
        density = nx.density(self.evo_graph)
        if task_type == "info_density":
            return torch.tensor(2.0 - density * 2.0, device=device)
        elif task_type == "error_rate":
            return torch.tensor(0.1 + (1 - density) * 0.1, device=device)
        elif task_type == "random_access":
            return torch.tensor(0.2 + density * 0.1, device=device)
        else:
            return torch.var(node_features)

    def compute_metrics(self, node_features, edge_index, edge_attr, tasks):
        G_nx = nx.from_edgelist(edge_index.t().cpu().numpy())
        info_density = nx.density(G_nx) * 2.0
        error_rate = 1 - nx.density(G_nx) * 0.1
        random_access = nx.density(G_nx) * 0.1
        constraints = torch.var(node_features).item()
        coherence = torch.mean(torch.abs(torch.corrcoef(node_features))).item()
        phi = self.compute_phi(G_nx)
        irreducibility = self.compute_irreducibility(G_nx)
        return {
            'info_density': info_density,
            'error_rate': error_rate,
            'random_access': random_access,
            'constraints': constraints,
            'coherence': coherence,
            'phi': phi,
            'irreducibility': irreducibility
        }

    def compute_phi(self, G_nx):
        nodes = list(G_nx.nodes())
        if len(nodes) < 2:
            return 0.0
        subsets = [nodes[:len(nodes)//2], nodes[len(nodes)//2:]]
        mutual_info = 0.0
        for subset in subsets:
            subgraph = G_nx.subgraph(subset)
            density = nx.density(subgraph)
            mutual_info += density * (1 - density)
        return mutual_info

    def compute_irreducibility(self, G_nx):
        return len(G_nx.edges()) / max(1, len(G_nx.nodes()))

    def calculate_fitness(self, metrics):
        norm_metrics = {}
        for metric, value in metrics.items():
            if metric in self.config.metric_ranges:
                min_val, max_val = self.config.metric_ranges[metric]["min"], self.config.metric_ranges[metric]["max"]
                norm_value = (value - min_val) / (max_val - min_val) if max_val > min_val else 0.5
                if not self.config.metric_directions[metric]:
                    norm_value = 1.0 - norm_value
                norm_metrics[metric] = norm_value
            else:
                norm_metrics[metric] = value
        return sum(self.config.metric_weights[m] * norm_metrics[m] for m in norm_metrics)

    def average_metrics(self, metrics_batch):
        return {m: np.mean([bm[m] for bm in metrics_batch]) for m in self.config.metrics}

    def evolve_graph(self, fitness_scores):
        best_idx = np.argmax(fitness_scores)
        best_fitness = fitness_scores[best_idx]
        for u, v in list(self.evo_graph.edges()):
            if np.random.random() < self.config.variant_configs["SynapticDEWE"]["mutation_rate"]:
                self.evo_graph[u][v]['weight'] += np.random.normal(0, 0.1)
                self.evo_graph[u][v]['weight'] = max(0.1, min(1.0, self.evo_graph[u][v]['weight']))
        for i in range(self.config.synaptic_nodes):
            for j in range(i + 1, self.config.synaptic_nodes):
                if np.random.random() < self.config.variant_configs["SynapticDEWE"]["crossover_rate"]:
                    if not self.evo_graph.has_edge(i, j):
                        self.evo_graph.add_edge(i, j, weight=np.random.uniform(0.1, 0.5))
        if best_fitness > self.best_fitness:
            self.best_fitness = best_fitness
            self.best_solution = {'graph': self.evo_graph.copy()}

    def save_checkpoint(self, path):
        checkpoint = {
            "best_fitness": self.best_fitness,
            "best_solution": {'graph': nx.to_dict_of_dicts(self.evo_graph)},
            "iteration": self.iteration,
            "config": vars(self.config)
        }
        torch.save(checkpoint, path)
        logger.info(f"Saved checkpoint to {path}")

    def load_checkpoint(self, path):
        checkpoint = torch.load(path)
        self.best_fitness = checkpoint["best_fitness"]
        self.best_solution = checkpoint["best_solution"]
        self.iteration = checkpoint["iteration"]
        self.evo_graph = nx.from_dict_of_dicts(checkpoint["best_solution"]["graph"])
        logger.info(f"Loaded checkpoint from {path}")

class DNADataset:
    """Synthetic Savannah dataset for DNA encoding"""
    def __init__(self, config):
        self.config = config
        self.data = []
        self.task_types = ["info_density", "error_rate", "random_access"]
        self._generate_synthetic_data()

    def _generate_synthetic_data(self):
        nucleotides = ['A', 'C', 'G', 'T']
        for i in range(10000):  # 10k sequences
            seq_length = np.random.randint(50, 300)
            sequence = ''.join(np.random.choice(nucleotides, size=seq_length))
            sequence = self.config.dna_spec.fix_sequence(sequence)  # Ensure HEDGES compatibility
            task_type = np.random.choice(self.task_types)
            self.data.append({"id": f"sample_{i}", "sequence": sequence, "type": task_type})

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        return self.data[idx]

class MSDEGA:
    """Optimized MSDEGA for Evo-2 1B finetuning with Savanna"""
    def __init__(self, config):
        self.config = config
        self.variants = []
        self.performance = {}
        self.selection_policy = AdaptiveSelectionPolicy(config)
        self.resource_allocator = ResourceAllocator(config)
        self.convergence_detector = ConvergenceDetector(config)
        self.metric_standardizer = MetricStandardization(config)
        self.iteration = 0
        self.logger = logging.getLogger("MSDEGA")

    def register_variant(self, variant_class, variant_config=None, name=None):
        if variant_config is None:
            variant_config = self.config.variant_configs.get(variant_class.__name__, {})
        for param in ["gc_min", "gc_max", "max_homopolymer", "target_density", "primer_length", "error_correction_method"]:
            variant_config[param] = getattr(self.config, param)
        variant = variant_class(variant_config)
        variant.name = name or f"{variant_class.__name__}_{len(self.variants)}"
        self.variants.append(variant)
        self.performance[variant.name] = {
            "fitness": [], "compute": [], "metrics": [], "selected": 0
        }
        self.logger.info(f"Registered variant: {variant.name}")

    def run(self, max_iterations=None):
        if max_iterations is None:
            max_iterations = self.config.max_iterations
        self.logger.info(f"Starting MSDEGA with {len(self.variants)} variants")
        for iteration in range(max_iterations):
            self.iteration = iteration
            self.logger.info(f"Iteration {iteration+1}/{max_iterations}")
            selected_variants = self.selection_policy.select(self.variants, self.performance)
            allocations = self.resource_allocator.allocate(selected_variants, self.performance)
            results = {}
            for variant, allocation in zip(selected_variants, allocations):
                self.logger.info(f"Running {variant.name} with allocation {allocation:.2f}")
                self.performance[variant.name]["selected"] += 1
                result = variant.run_iteration(allocation)
                results[variant.name] = result
            for name, result in results.items():
                self.performance[name]["fitness"].append(result["fitness"])
                self.performance[name]["compute"].append(result["compute_used"])
                self.performance[name]["metrics"].append(result.get("metrics", {}))
            self.metric_standardizer.update_metric_ranges({name: [r["metrics"]] for name, r in results.items()})
            if iteration % 3 == 0:
                self._evolve_variants()
            if iteration % self.config.checkpoint_frequency == 0:
                self._save_checkpoint()
            if self.convergence_detector.has_converged(self.performance):
                self.logger.info(f"Convergence at iteration {iteration+1}")
                break
        self._save_checkpoint()
        best_result = self._get_best_result()
        if self.config.generate_analysis:
            self.analyze_results()
        return best_result

    def _evolve_variants(self):
        variant_fitness = []
        for variant in self.variants:
            perf = self.performance[variant.name]
            if perf["fitness"]:
                fitness = np.mean(perf["fitness"][-3:])
                variant_fitness.append((variant, fitness))
        if len(variant_fitness) < 2:
            return
        variant_fitness.sort(key=lambda x: x[1], reverse=True)
        top_variants = variant_fitness[:len(variant_fitness)//2]
        for i in range(min(2, len(top_variants))):
            idx1, idx2 = np.random.choice(len(top_variants), 2, replace=False)
            parent1, _ = top_variants[idx1]
            parent2, _ = top_variants[idx2]
            if np.random.random() < self.config.meta_crossover_rate:
                new_variant = self._crossover_variants(parent1, parent2)
                name = f"Cross_{parent1.name}_{parent2.name}_{self.iteration}"
                self.register_variant(new_variant.__class__, new_variant.config, name)
        for i in range(min(1, len(top_variants))):
            idx = np.random.choice(len(top_variants))
            parent, _ = top_variants[idx]
            if np.random.random() < self.config.meta_mutation_rate:
                new_variant = self._mutate_variant(parent)
                name = f"Mutate_{parent.name}_{self.iteration}"
                self.register_variant(new_variant.__class__, new_variant.config, name)

    def _crossover_variants(self, parent1, parent2):
        config1 = parent1.config
        config2 = parent2.config
        new_config = {}
        common_params = set(config1.keys()).intersection(set(config2.keys()))
        for param in common_params:
            value1 = config1[param]
            value2 = config2[param]
            if isinstance(value1, (int, float)) and isinstance(value2, (int, float)):
                crossover_type = np.random.choice(["uniform", "blend"])
                if crossover_type == "uniform":
                    new_config[param] = value1 if np.random.random() < 0.5 else value2
                else:
                    alpha = np.random.uniform(-0.3, 1.3)
                    new_value = value1 + alpha * (value2 - value1)
                    if isinstance(value1, int) and isinstance(value2, int):
                        new_value = int(round(new_value))
                    new_config[param] = new_value
            else:
                new_config[param] = value1 if np.random.random() < 0.5 else value2
        for param in set(config1.keys()) - common_params:
            if np.random.random() < 0.5:
                new_config[param] = config1[param]
        for param in set(config2.keys()) - common_params:
            if np.random.random() < 0.5:
                new_config[param] = config2[param]
        return SynapticDEWE(new_config)

    def _mutate_variant(self, parent):
        config = copy.deepcopy(parent.config)
        for key, value in config.items():
            if isinstance(value, (int, float)):
                if isinstance(value, int):
                    if value != 0:
                        perturbation = int(value * np.random.uniform(-0.2, 0.2))
                        config[key] = max(1, value + perturbation)
                    else:
                        config[key] = np.random.choice([-1, 0, 1])
                else:
                    perturbation = value * np.random.uniform(-0.2, 0.2)
                    config[key] = value + perturbation
        return SynapticDEWE(config)

    def _save_checkpoint(self):
        checkpoint_dir = os.path.join(self.config.output_dir, f"checkpoint_{self.iteration}")
        os.makedirs(checkpoint_dir, exist_ok=True)
        meta_state = {
            "iteration": self.iteration,
            "performance": self.performance,
            "exploration_rate": self.selection_policy.exploration_rate
        }
        with open(os.path.join(checkpoint_dir, "meta_state.json"), "w") as f:
            json.dump(meta_state, f, indent=2)
        for variant in self.variants:
            variant_path = os.path.join(checkpoint_dir, f"{variant.name}.pt")
            variant.save_checkpoint(variant_path)
        self.logger.info(f"Saved checkpoint to {checkpoint_dir}")

    def _get_best_result(self):
        best_name = None
        best_fitness = -float('inf')
        best_solution = None
        for variant in self.variants:
            solution = variant.best_solution
            fitness = variant.best_fitness
            if fitness > best_fitness:
                best_fitness = fitness
                best_solution = solution
                best_name = variant.name
        return {
            "name": best_name,
            "fitness": best_fitness,
            "solution": best_solution
        }

    def analyze_results(self):
        meta_state_path = os.path.join(self.config.output_dir, f"checkpoint_{self.iteration}", "meta_state.json")
        if not os.path.exists(meta_state_path):
            self.logger.warning("No final checkpoint for analysis")
            return
        with open(meta_state_path, "r") as f:
            meta_state = json.load(f)
        performance = meta_state["performance"]
        plt.figure(figsize=(12, 8))
        plt.subplot(2, 2, 1)
        for variant, perf in performance.items():
            if perf["fitness"]:
                plt.plot(perf["fitness"], label=variant)
        plt.title("Fitness Evolution")
        plt.xlabel("Iteration")
        plt.ylabel("Fitness")
        plt.legend()
        plt.subplot(2, 2, 2)
        variants = list(performance.keys())
        selections = [perf["selected"] for perf in performance.values()]
        plt.bar(variants, selections)
        plt.title("Selection Frequency")
        plt.xlabel("Variant")
        plt.ylabel("Times Selected")
        plt.xticks(rotation=45)
        plt.subplot(2, 2, 3)
        for variant, perf in performance.items():
            if perf["fitness"] and perf["compute"]:
                efficiency = [f/c for f, c in zip(perf["fitness"], perf["compute"])]
                plt.plot(efficiency, label=variant)
        plt.title("Compute Efficiency")
        plt.xlabel("Iteration")
        plt.ylabel("Fitness / Compute")
        plt.legend()
        best_variant = max(performance.items(), key=lambda x: max(x[1]["fitness"]) if x[1]["fitness"] else -float('inf'))[0]
        plt.subplot(2, 2, 4)
        metrics = performance[best_variant]["metrics"]
        if metrics:
            for metric in set().union(*[m.keys() for m in metrics]):
                values = [m.get(metric, 0) for m in metrics]
                plt.plot(values, label=metric)
            plt.title(f"Metrics for {best_variant}")
            plt.xlabel("Iteration")
            plt.ylabel("Value")
            plt.legend()
        plt.tight_layout()
        plt.savefig(os.path.join(self.config.output_dir, "performance_analysis.png"))
        plt.close()
        self.logger.info(f"Analysis saved to {self.config.output_dir}/performance_analysis.png")
