IX. CONTROLLER LOGIC v3

Core selection pipeline:

valid_moves = []

for move in MOVE_LIBRARY:
    if move.phase != current_phase: continue
    if user_action_curr not in move.preconditions.user_actions: continue

    lo, hi = move.preconditions.progress_range
    if not (lo <= progress <= hi): continue

    if move.preconditions.trap_active != trap_state: continue

    valid_moves.append(move)

move = weighted_sample(valid_moves)
variant = random.choice(move.phrase_family)


Then:

Apply move

Log n3_meta.reason

Update progress

Shift phase if needed

Apply move_chain if user reacts as predicted

This maintains human-like pedagogy.
